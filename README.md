# PDF Görsel Ayıklayıcı

Modern ve kullanıcı dostu bir web arayüzü ile PDF dosyalarından görselleri ayıklayın.

## Özellikler

- 🖼️ Tüm görselleri otomatik ayıklar
- 📦 ZIP dosyası olarak indirebilme
- 👁️ İlk 20 görselin önizlemesi
- 💻 Lokal olarak çalışır (internet gerekli değil)

## Kurulum

1. Gerekli paketleri yükleyin:
```bash
pip install -r requirements.txt
```

## Kullanım

Uygulamayı başlatın:
```bash
python pdf_image_extractor_ui.py
```

Tarayıcınızda otomatik olarak açılacaktır (varsayılan: http://127.0.0.1:7860)

## Gereksinimler

- Python 3.8+
- PyMuPDF (fitz)
- Gradio
- Pillow
